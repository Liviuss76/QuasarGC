QuasarGC
========

Game Center Java Server
Standalone build setup of a spring boot application with embedded Tomcat 7 with thymeleaf frontend and postgresql backend in a self hosting .jar file built by maven.

* Spring Boot 
* Spring Security
* Spring MVC
* JPA
* Hibernate
* Tomcat 7
* Thymeleaf
* Bootstrap
* PostgresSQL 9.3

###### Build with Maven

```
	mvn clean package
```

###### Run

```
	mvn clean package
```
